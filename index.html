Alert

The alert component has basic 4 types, alert-success, alert-danger, alert-info, alert-warning, and 2 additional basic types alert-primary and alert-secondary.

Avatar

The avatar component has basic 5 sizes, avatar-xl, avatar-lg, avatar-md, avatar-sm, avatar-xs, and by using round we can have round avatars and square for square avatars.

Badge

The badge component has basic 4 types, badge-avatar-large, badge-avatar-medium, badge-avatar-small, badge-icon-small, and 2 status, online and offline.

Button

The Button component has basic 2 types, solid buttons and outlined buttons.

Solid Buttons

The Solid button components has basic 5 types, btn-primary, btn-round, btn-link, btn-hover, btn-disabled.

Outlined Buttons

The Outlined button components has basic 4 types, btn-outline-primary, btn-outline-round, btn-outline-icon, btn-outline-hover, btn-outline-disabled.

Icon and Floating Action Buttons

The floating action button can be implemented using btn-floating-action, and a button with an icon can be implemented using btn-icon.
